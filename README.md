# RDNS V2.7 - Master Edition 🚀
Tool multi-fungsi untuk Geolocation, Metadata Stripping, dan Network Scanning yang dirancang khusus untuk pengujian keamanan (Penetration Testing) dan edukasi.

## 🛠️ Fitur Utama
- **📍 Geolocate:** Melacak koordinat GPS target secara akurat melalui pancingan link browser dengan dukungan multi-tunnel (Cloudflare, Serveo, Localhost).
- **🖼️ Metadata:** Analisis data tersembunyi (EXIF) pada gambar dan membersihkannya secara total untuk menjaga privasi.
- **🔍 Specter:** Scanner port jaringan cepat untuk melihat celah keamanan pada IP atau Domain target.
- **📁 Archive:** Manajemen log hasil sniffing yang rapi dengan sistem RWD (Read, Write, Delete).
